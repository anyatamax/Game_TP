1) В игре существует два базовых типа: Персонаж и Монстр. В начале игры игрок выбирает персонажа. На выбор ему предоставляется: рыцарь, обычный человек, маг и принцесса. Каждый из типо персонажей имеет свои привелегии в игре. При выборе персонажа создается фабрика, которая по id юнита создает требуемый объект. Все классы юнитов наследники абстрактного класса Unit.
Класс монстр включает в себя структуру со всеми возможными типами монстров и их характеристиками. Один монстр создается при сталкновении игрока с монстром в ходе игры. Тогда игроку выдается рандомный монстр, которого сгенерирует программа.

Во втором чекпоинте было добавлено: (файл Game/ComposeUnits.h)
2) Игрок перед тем, как начать игру собирает свою "команду" персонажей. Для этого просто вводит номера тех юнитов, с помощью которых хочет играть. Тогда отдельный класс создает объект в виде вектора из юнитов, которых выбрал игрок. Далее именно этим объектом будет управлять игрок в ходе игры.
